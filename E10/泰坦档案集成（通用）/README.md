# 泰坦档案集成（通用）

E10集成泰坦档案，此为档案通用开发，可适配其它档案厂商。

功能：

- 与标准功能集成，使用标准功能的档案集成，归档方案为ecdoe自定义归档

- 采用配置化开发，可配置推送流程字段、档案接口配置、档案包打包配置，尽可能通用

- 与流程集成，流程归档后可推送到档案系统

- 可在台账中查看档案推送记录与结果

- 可通过档案系统调用OA接口，将归档结果反馈到OA
