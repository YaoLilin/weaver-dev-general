# 量子档案集成

E9集成量子档案，可将流程数据推送到档案系统。采用配置化开发，可部署到其它系统。

功能：

1. 配置化开发，基本所有推送字段都可在建模中进行配置，未来新增推送字段或修改字段取值无需修改代码，要上传到档案系统的文件也可在建模中配置。
2. 所有流程的档案推送结果都可以在【档案推送结果】建模中查看，支持手动重新推送
3. 支持历史流程批量推送，需要在档案推送结果查询中的右上角点击历史流程批量推送按钮（此按钮需要自己配，配置方法详见ecode应用中的readme.md文件），选择要推送历史数据的流程，然后点推送即可。
4. 支持将表单字段中的word文档转换成PDF
5. 档案系统可调用OA接口将档案归档结果传到OA，接口地址：OA服务器地址/api/secondev/archive/status
6. 可以使用计划任务，定时查询档案归档状态，更新到档案推送结果台账中，计划任务类：com.customization.yll.wuling.archive.job.ArchiveStatusCheckJob
